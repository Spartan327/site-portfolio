# site-portfolio

Данный проект является портфолио, для графического дизайнера.

На текущий момент реализовано:
- главная страница;
- галерея работ;
- вывод работ по тегу;
- добавление работы;
- редактирование работы;
- авторизация зарегистрированных пользователей;
- авторизация через соц.сеть вконтакте.
